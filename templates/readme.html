サイト案内

映画のオンラインチケット購入サイトです。購入の大まかな流れは会員登録→映画詳細ページ→座席選択→チケット購入の順番です。
会員が作成されると始めに50000ポイント所持しているので、そのポイントを消費して映画チケットを購入出来ます。
会員情報はポイントを含めマイページから変更出来るので、ポイント残高が足りなくなった場合は追加して下さい。
username : debug password : debug テスト用のユーザーですが此方を使用すると会員登録しなくてもログイン出来ます。
